A detailed reference to the Yao internal processes.

## Internal Processes

| Documentation                                    | Namespace                         | Description                                                               |
| ------------------------------------------------ | --------------------------------- | ------------------------------------------------------------------------- |
| [Model Process](processes/models)                | models.<model_widget_id>.\*       | Interacts with the database to perform CRUD operations on models.         |
| [Schema Process](processes/schemas)              | schemas.<connector_widget_id>.\*  | Manages the schema structure and database migrations.                     |
| [Store Process](processes/stores)                | stores.<store_widget_id>.\*       | Provides key-value store operations for storing data.                     |
| [Flow Process](processes/flows)                  | flows.<flow_widget_id>            | Executes and manages process flows within the application.                |
| [Schedule Process](processes/schedule)           | schedules.<schedule_widget_id>.\* | Defines scheduled tasks and periodic executions.                          |
| [Pipe Process](processes/pipe)                   | pipes.<pipe_widget_id>.\*         | Handles data flow between processes, such as streaming or piping.         |
| [Pipe Process](processes/pipe)                   | pipe.\*                           | Provides utility processes for handling data pipes and flows.             |
| [File System Process](processes/fs)              | fs.<namespace>.\*                 | Manages file system interactions, including reading and writing.          |
| [API Process](processes/api)                     | api.\*                            | Utility processes for managing and interacting with APIs.                 |
| [HTTP Process](processes/http)                   | http.\*                           | Processes HTTP requests and responses.                                    |
| [Session Process](processes/session)             | session.\*                        | Manages user sessions and state information.                              |
| [Encoding Process](processes/encoding)           | encoding.\*                       | Provides encoding and decoding utilities for various formats.             |
| [Crypto Process](processes/crypto)               | crypto.\*                         | Performs cryptographic operations, such as hashing and encryption.        |
| [SSL Process](processes/ssl)                     | ssl.\*                            | Handles SSL/TLS encryption for secure communications.                     |
| [Utils Process](processes/utils)                 | utils.\*                          | Provides general utility functions for various tasks.                     |
| [OpenAI Process](processes/openai)               | openai.\*                         | Interacts with OpenAI services for AI-powered tasks.                      |
| [MOAPI Process](processes/moapi)                 | moapi.\*                          | Processes related to MOAPI services for integrating APIs.                 |
| [Neo Process](processes/neo)                     | neo.\*                            | Manages Neo chatbot interactions and AI-driven conversations.             |
| [SUI Process](processes/sui)                     | sui.\*                            | Provides utilities for the SUI engine and web page management.            |
| [Yao App Process](processes/yao-app)             | yao.app.\*                        | Processes for managing the Yao application widget and logic.              |
| [Yao Login Process](processes/yao-login)         | yao.login.\*                      | Handles user authentication and login processes.                          |
| [Yao Table Process](processes/yao-table)         | yao.table.\*                      | Manages table widget operations, including data binding and CRUD.         |
| [Yao Form Process](processes/yao-form)           | yao.form.\*                       | Handles form widget operations, including data validation and submission. |
| [Yao List Process](processes/yao-list)           | yao.list.\*                       | Manages dashboard widgets and displays list-based data.                   |
| [Yao Chart Process](processes/yao-chart)         | yao.chart.\*                      | Manages chart widgets and data visualization in the admin panel.          |
| [Yao Dashboard Process](processes/yao-dashboard) | yao.dashboard.\*                  | Manages dashboard widgets for creating custom admin panels.               |
| [Script Process](processes/scripts)              | scripts.\*                        | Custom processes written in TypeScript to extend functionality.           |
| [Plugin Process](processes/plugins)              | plugins.\*                        | Manages GRPC plugin integration and custom plugin logic.                  |
